## Instalação

#### Esse projeto usa Node.js e Yarn
- clone o projeto
  ```
  git clone https://github.com/paulopbomfim/nlw1-ecoleta.git
  ```
- Entre na pasta do projeto usando o terminal e digite
  ```
  yarn
  ```
- Em seguida digite
  ```
  yarn start
  ```
